## Quick Setup Instructions

1. **Go to**: https://programmablesearchengine.google.com/controlpanel/create
2. **Name**: Enter "careerist-job-search" or "careerist"
3. **Add Sites**: Copy each line above and paste into "Enter a site or pages" field
4. **Click "Add"** after each site
5. **Settings**: 
   - Image search: OFF
   - SafeSearch: OFF
6. **Create**: Check reCAPTCHA and click "Create"

## Priority Order

**Start with Tier 1** (8 sites) - these are the most important
**Then add Tier 2** (8 sites) - tech-specific
**Then add Tier 3** (12 sites) - regional coverage
**Then add Tier 4-6** as needed

**Note**: Google allows maximum 50 distinct domains. Prioritize Tier 1-3 first.

## Alternative: Multiple Search Engines

If you need more than 50 domains, create multiple search engines:
- **Search Engine 1**: Global job boards (Tier 1-2)
- **Search Engine 2**: Regional boards (Tier 3)
- **Search Engine 3**: Niche boards (Tier 4-6)

Then use all three in your application.

## Testing Your Setup

After adding sites:
1. Go to "Control Panel" → "Test"
2. Search for: "software engineer"
3. Verify results come from job sites
4. Check result quality

## Site Format Notes

✅ **Correct formats**:
- `www.example.com/jobs/*` - Searches all job pages
- `www.example.com/careers/*` - Searches career pages
- `careers.example.com/jobs/*` - Searches subdomain

❌ **Incorrect formats**:
- `*.example.com` - Not allowed (public suffix)
- `example.com/*` - May not work (use www.)
- `example.com` - Too broad, use specific path
