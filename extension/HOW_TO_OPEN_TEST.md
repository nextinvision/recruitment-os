# How to Open test/popup-test.html

## ❌ Don't Search on Google!

The file `test/popup-test.html` is a **local file** in your project, not a website. You cannot search for it on Google.

## ✅ Correct Ways to Open It

### Method 1: Double-Click (Easiest)

1. Open File Explorer
2. Navigate to: `C:\Users\anand\Documents\NEXTIN VISION\Recruitment-os\extension\test\`
3. Find `popup-test.html`
4. **Double-click** the file
5. It will open in your default browser

### Method 2: Right-Click → Open With

1. Navigate to the file in File Explorer
2. **Right-click** on `popup-test.html`
3. Select **"Open with"** → **"Google Chrome"** (or your preferred browser)

### Method 3: Drag and Drop

1. Open Chrome browser
2. Open File Explorer
3. **Drag** `popup-test.html` from File Explorer
4. **Drop** it into the Chrome window

### Method 4: Use File Path in Browser

1. Open Chrome
2. In the address bar, type:
   ```
   file:///C:/Users/anand/Documents/NEXTIN%20VISION/Recruitment-os/extension/test/popup-test.html
   ```
3. Press Enter

### Method 5: From VS Code

1. In VS Code, right-click on `test/popup-test.html`
2. Select **"Open with Live Server"** (if you have the extension)
3. OR select **"Reveal in File Explorer"** then double-click

## Troubleshooting

### File Not Found?
- Make sure you're in the `extension` folder
- Check that `test/popup-test.html` exists
- Run: `ls test/popup-test.html` (or `dir test\popup-test.html` on Windows)

### Page Not Loading?
- Make sure you opened it as a file (file://), not searching on Google
- Try a different browser
- Check browser console for errors (F12)

### Still Confused?
The file is like opening a Word document - you open it from your computer, not from the internet!
