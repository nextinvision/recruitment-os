# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Quick Fix: Create Placeholder Icons

For testing, you can create simple placeholder icons using any image editor or online tool.

## Temporary Solution

The manifest has been updated to make icons optional. The extension will work without icons (Chrome will show a default icon).
